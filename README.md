# Projeto-PIM---An-lise-e-Desenvolvimento-de-Sistemas
Sistema acadêmico desenvolvido em Python como projeto PIM do curso de Análise e Desenvolvimento de Sistemas (UNIP). O sistema centraliza a gestão de alunos, turmas, notas, frequência, atividades e comunicação entre usuários, aplicando conceitos de engenharia de software, UML e arquitetura cliente-servidor para otimizar processos acadêmicos.
